# 侯捷C++高级编程课程

将构造函数放在private中，可以形成Singleton模式，就只有创建了一个对象。

在函数的**()**与**{}**之间加上const表示：函数不会改变数据。

```C++
class A
{
	double test() const {return x;}
	private:
		int x;
};
```

C++的引用在底层设计等于C语言的指针。const可以避免引用的修改。
